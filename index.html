Featured Project

Price Engine Re-Architecture

This project involved re-architecting a traditional monolithic pricing engine into a scalable microservices-based solution deployed on AWS. Key aspects included:

  • Containerizing microservices using Docker and deploying to ECS Fargate
  • Using AWS Load Balancer and Route53 for routing and domain management
  • Infrastructure provisioning via Terraform and Ansible, with remote state stored in S3
  • CloudWatch Agent setup on EC2 database instances (ClickHouse, YugabyteDB) for metric collection
  • Grafana and Prometheus integration for performance visualization
  • API gateway setup using Kong for external API call tracking
  • CI/CD pipeline using GitHub Actions for automated deployment
